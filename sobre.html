Sobre mim

Com uma paixão por transformar ideias em soluções digitais, eu me dedico a criar experiências de usuário inovadoras e eficientes. Utilizo as linguagens essenciais do desenvolvimento web — HTML, CSS e JavaScript — como minhas principais ferramentas para construir interfaces dinâmicas e funcionais. Minha abordagem vai além do código.

Eu acredito que a chave para um bom projeto é a resolução criativa de problemas, combinando estética e funcionalidade para entregar produtos que realmente fazem a diferença. Estou em busca da minha primeira oportunidade no mercado de trabalho como jovem aprendiz ou estágio, além desafios e oportunidades para aprimorar minhas habilidades e contribuir para projetos que possam mudar vidas.

Projetos Pessoais

Relogio Digital

A estrutura HTML e o visual CSS cuidam do design, fazendo o display parecer um painel digital, com luzes e a fonte certa. A parte mais inteligente, feita com JavaScript, garante que a hora mostrada esteja sempre certa, atualizando o horário a cada segundo automaticamente. É a prova de que a aplicação sabe funcionar e se manter sincronizada.

Calculadora Relacional

Este projeto é diferente de uma calculadora aritmética simples. O HTML e o CSS são utilizados para montar uma interface limpa, onde o usuário insere uma expressão de comparação (ex: 10 > 5 ). O JavaScript é responsável por: Analisar a string de entrada para identificar os dois operandos e o operador relacional (==, >, <=, etc.); Executar a comparação ; Apresentar o resultado final de forma clara, retornando TRUE ou FALSE.

Crud

Este é um sistema completo para organizar informações, como uma lista de tarefas, que permite adicionar, ler, editar e apagar itens. O visual HTML e CSS permite que você veja a lista e tenha botões para as ações. O JavaScript faz a gestão de tudo por trás: ele salva sua lista no computador para que ela não se perca quando você fechar a página, e permite que você a atualize (marcando como concluída) ou a exclua. Mostra a capacidade de controlar e guardar dados..
